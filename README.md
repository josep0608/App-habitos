HabitFlow - Aplicación de Gestión de Hábitos
HabitFlow es una aplicación diseñada para ayudar a los usuarios a gestionar y mejorar sus hábitos diarios. Con ella puedes registrar tus tareas, ver estadísticas de tu progreso y obtener gráficos para visualizar tus logros. Además, incluye un sistema de suscripción premium que desbloquea funciones adicionales.
Características Principales
Gestión de Tareas: Crea, edita y completa tareas diarias.
Estadísticas y Gráficos: Visualiza tu progreso con gráficos interactivos.
Sistema Premium: Accede a funciones exclusivas para usuarios premium.
Exportación de PDF: Descarga un informe en PDF con tus tareas cumplidas y atrasadas.
Asistente AI: Responde preguntas y obtén recomendaciones personalizadas.
Requisitos Previos
Node.js v14.17.0 o superior
MySQL para la base de datos
Android Studio o Xcode para desarrollo móvil
Expo CLI para desarrollo con React Native
